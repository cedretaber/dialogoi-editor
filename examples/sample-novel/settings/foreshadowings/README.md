# 伏線管理

物語の伏線を管理するディレクトリです。

## 管理方針

- 各伏線ファイルには埋蔵位置（start）と回収位置（goal）を明記
- 伏線の種類や重要度をタグで分類
- 回収予定の章や内容を詳細に記載

## 伏線の種類

### 短期伏線
- 数章以内で回収される伏線
- キャラクターの行動や会話に関連

### 中期伏線
- 複数の章にまたがる伏線
- 物語の展開に影響を与える要素

### 長期伏線
- 物語全体に関わる大きな伏線
- 主人公の出自や世界の秘密など

## ファイル構成

- **hidden_power.md** - 主人公の隠された力
- **academy_secret.md** - 魔法学院の秘密
- **ancient_prophecy.md** - 古代の予言
